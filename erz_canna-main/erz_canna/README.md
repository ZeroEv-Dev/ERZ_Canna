# ERZ_CANNA

ROLLA E FUMA CANNA

## INSTALLAZIONE
- INSERISCI IN `[esx]`
- AGGIUNGI QUESTO NEL TUO `server.cfg`:

```
start erz_canna
```

## ITEM
IMPORTA IL FILE `items.sql` NEL TUO DATABASE

SE USI OX_INVENTORY:
	['cimetta'] = {
		label = 'Cimetta',
		weight = 10,
	},

	['cartine'] = {
		label = 'Cartine Lunghe',
		weight = 100,
	},

    ['accendino'] = {
		label = 'Accendino',
		weight = 150,
	},

    ['canna'] = {
		label = 'Canna',
		weight = 200,
	},
}
